<h1> Repository used to manage zsh scripts</h1>

<h2>createc.sh</h2>
<p>This script is used to automatize the creation of c project with dir structure show below</p>

├── _include
│   └── **.h
|
├── _src
│   ├── main.c
│   └── **.c
|
├── Makefile
<br>
<h4>The command structure is</h4>
```createc.sh -p [projectname] -l [\"your_libraries\"] ```
